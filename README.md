# DerpLua v1

DerpLua is a simple visual Roblox Lua behavior builder.

## What v1 does

- Add behavior blocks
- Choose an event
- Edit block settings
- Generate copyable Roblox Lua
- Save/load locally
- Export/import project JSON

## Best first uses

- Touch damage parts
- Fling pads
- Random color parts
- Moving/rotating parts
- ProximityPrompt scripts
- ClickDetector scripts
- Tool activated scripts

## How to use on GitHub Pages

Upload these files to a repo:

- `index.html`
- `style.css`
- `main.js`

Then enable GitHub Pages.
